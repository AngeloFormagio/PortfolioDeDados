Análise de Dados de Vendas e Perfil de Clientes em Shopping Malls

Descrição do Projeto: Este projeto explora um conjunto de dados de vendas e clientes provenientes de uma rede de shopping malls na Califórnia. O objetivo foi extrair insights sobre padrões de compras, perfil de clientes e comportamento de vendas nos diferentes shoppings, aplicando técnicas de análise exploratória, modelagem preditiva

Análise Integrada entre Clientes e Vendas:

Foi realizada uma junção entre os dados de clientes e vendas, proporcionando uma visão detalhada do comportamento de compra por idade, gênero e métodos de pagamento. Gasto médio por faixa etária e preferências de categorias foram analisados, revelando perfis de consumo e segmentos de maior valor agregado.

Ferramentas e Tecnologias Utilizadas:

  • Python: Manipulação de dados e visualizações.
  • Pandas e NumPy: Análise exploratória e preparação dos dados.
  • Matplotlib e Seaborn: Visualizações de dados e clusters.
  • Scikit-Learn: RegressãoLinear

Análise de Dados Exploratória sobre um conjunto de dados referente a adultos, escolaridade, salario e país

Descrição do Projeto: Este projeto explora uma base de dados com informações sobre a população adulta, explorando dados sobre a idade média dos homens, remuneração para pessoas com e sem diploma, horas trabalhadas em realação ao salário, país com mais pessoas bem remuneradas, suas porcentagens e dados em geral.

Análise entre diplomas e remuneração:

Realizada a verificação do número e posteriormente porcentagem de pessoas com e sem alta escolaridade que recebem mais que determinado valor.

Análise entre profissão e remuneração

Verificamos qual país do dataset tem um maior número de pessoas bem remuneradas e qual a profissão mais comum entre essas pessoas.

Ferramentas e Tecnologias Utilizadas:

  • Python: Manipulação de dados e visualizações.

Análise de Dados Exploratória sobre os melhores 50 exercícios físicos

Descrição do Projeto: Este projeto explora uma base de dados de exercícios físicos, suas repetições, seus sets e calorias queimadas.

Queima de Calorias X exercícios:

Após ser feito o tratamento dos dados lidando com valores faltantes foi realizada a verificação dos melhores exercícios para a queima de calorias juntamente com a visualização desses dados em gráficos.

Separamos os exercícios em grupos muscular para saber qual o grupo muscular que tende a queimar mais calorias

Plotamos gráficos em sets, reps, e queima de caloria para visualizar a correlação desses atributos

Ferramentas e Tecnologias Utilizadas:

  • Python: Manipulação de dados e visualizações.
  • Pandas e NumPy: Análise exploratória e preparação dos dados.
  • Matplotlib e Seaborn: Visualizações de dados e correlações
